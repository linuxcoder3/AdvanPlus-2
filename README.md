# botzedlink
فایل را از حالا زیپ خارج کرده و به مسیر زیر برید :
./data/config.lua
اطلاعات خواسته شده را جایگزین کنید.

cd AdvanPlus
chmod +x lunch
./lunch install
./lunch cli
شماره ربات را وارد کنید.
برای خاموش نشدن ربات به اتولانچ احتیاج دارید :

cd AdvanPlus
cd Cli
screen ./lunch cli
و برای اتولانچ api
cd AdvanPlus
cd Cli
screen ./lunch api
